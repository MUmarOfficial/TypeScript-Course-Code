# Diagram (خاکہ) to Explain Working

Imagine this game as a journey:

```
+----------------+
|      Start     |
| (Game Shuru!)  |
+----------------+
        |
        V
+-----------------------------------+
|   Computer sochega aik "Secret"   |
|   Random Number (1 se 100 tak)    |
|   (Example: 42) (ایک خفیہ نمبر)  |
+-----------------------------------+
        |
        V
+-----------------------------------+
|  Guesses Count = 0 (اندازے گننا)  |
+-----------------------------------+
        |
        V
+------------------------------------------------------+
|               Loop Shuru (Bar Bar Chalo!)            |
|  (Jab tak aapka guess sahi nahi hota, chalta rahega) |
+------------------------------------------------------+
        |
        V
+-----------------------------------+
|  Aap se poochhenge "Guess karo?"  |
|  (Example: Aapne kaha 50)         |
+-----------------------------------+
        |
        V
+-----------------------------------+
|   Guesses Count ko 1 se badhao    |
|     (Count++) (گنتی بڑھاؤ)      |
+-----------------------------------+
        |
        V
+-----------------------------------+
|    Aapka guess aik number hai?    |
|   (Valid number check - نمبر ہے؟) |
+-----------------------------------+
        | No           | Yes
        V              V
+-----------------+   +---------------------------------------+
|   "Sahi number  |   |   Aapka guess Secret Number se bada hai? |
|   do please!"   |<--|   (Is 50 > 42?) (بڑا ہے؟)            |
| (Phir se poocho) |   +---------------------------------------+
+-----------------+   | Yes                  | No
        ^             V                      V
        |    +-----------------+   +---------------------------------------+
        |    | "Chota number   |   |   Aapka guess Secret Number se chota hai? |
        |    | batao!"         |   |   (Is 50 < 42?) (چھوٹا ہے؟)           |
        |    +-----------------+   +---------------------------------------+
        |                                 | Yes                   | No
        +---------------------------------+                       V
        |                                               +---------------------------------------+
        |                                               |   Aapka guess Secret Number ke barabar hai? |
        |                                               |   (Is 50 == 42?) (برابر ہے؟)          |
        |                                               +---------------------------------------+
        |                                                       | Yes                    | No
        |                                                       V                        V
        +-------------------------------------------------+   +-----------------+   +-----------------+
                                                        |   | "Mubarak ho!!" |   |  (Loop wapas)   |
                                                        |   |  (Game Jeet gaye!) |   |                 |
                                                        |   +-----------------+   +-----------------+
                                                        |           |
                                                        V           |
+-------------------------------------------------------+           |
|  Loop Khatam (Jab guess sahi ho jaye) (چکر ختم!)     |<----------+
+-------------------------------------------------------+
        |
        V
+-----------------------------------------------+
|   Batao: "Aapne X guesses mein jeet lia!"     |
|   (Display "You guessed in X guesses")       |
+-----------------------------------------------+
        |
        V
+-----------------------------------------------+
|  Game interface band karo (readline.close())  |
|  (Khalas!)                                    |
+-----------------------------------------------+
        |
        V
+----------------+
|       End      |
|  (Game Khatam!)|
+----------------+
```
